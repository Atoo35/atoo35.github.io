IM (Instant Messenger)

Built using MERN stack I wanted to explore Fullstack development. The project draws inspiration from Snapchat (which basically means I was lazy enough not to store the messages in the database).
Anyway it uses sockets.io to communicate with other users simultaneously. I have also implemented a mechanism where-in if person A texts B and person B is texting to person C, person B would see person A's name appearing on the left hand side menu indicating A is texting him/her.
Notifications aren't implemented as of yet.

NOTE:This project is still not complete so please ignore the UI.


How to run?

1. Navigate to frontend>src>services>api.js and change the IP address to your IP.
2. Similarly Navigate to frontend>src>pages>Home>index.js and change the IP on line 25.
3. In the folder backend, create a .env file and put values for MONGODB_URI and SECRET.
4. Open 2 cmd's, one for frontend and one for backend and go into the root folder of the respective folders in cmd.
5. In frontend cmd type 'npm run dev' and for backend type 'npm start'.
