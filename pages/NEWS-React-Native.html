News app built with React Native

This is a hybrid app built with React Native. It uses data from NewsAPI which includes local news, national news, categorical news or even news by news source.

There are 3 main pages in this app. The first is the home page which gives you the national news according to your location.

Second tab contains the alphabetical list of all the news sources available (This is hardcoded for the time being). Clicking on any of the tiles you will get a list of news sources starting with the selected alphabet.

The third tab includes the search bar by which you can search for desired news and even apply some fiters.
