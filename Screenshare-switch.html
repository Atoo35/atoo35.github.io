Screenshare and Screenswitch using OpenCV

This is among one of my favourite projects. It is actually usefull for people having a dual screen setup. The first version was made for a display connected with an HDMI cable but later on I developed it further to include screens in a LAN over Wi-fi.
It simply detects the red object in my hand shown in the video and depending upon the speed by which I move my hand it decides whether to switch the current window on another screen or to share the screen altogether


How to run?

It simply requires python and a few libraries to run it. The instructions are given in the Readme.md file in the repo. You can find the link to the repo below.
