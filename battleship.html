Battleship using c++

This is one of my earliest projects in college life. This was made as a part of Second Year Project. The language used for its creation is c++.

Battleship is a popular 2 player game wherein you have to hide your ships and turn by turn guess the location of the ships of the other player. In this particular project we have included only 2 ships per person of a fixed length to be hidden in the ocean of a 5x5 square.

It was basically built in linux and hence uses some linux specific system commands. Changing those system commands to their windows counter-parts would solve the issue.


How to run?

Open a command prompt and write the following one after the other
1. g++ battleship.cpp
2. a
